# Microsoft Fabric Tools Workload - v2025.12

**Release Date:** December 16, 2025

## Overview

December 2025 release of the Microsoft Fabric Tools Workload, a community collection of tools built on top of the official [Microsoft Fabric Extensibility Toolkit](https://github.com/microsoft/fabric-extensibility-toolkit).

## üéØ Major Updates

### Toolkit Update to v2025.12
- **Framework Upgrade**: Updated to Microsoft Fabric Extensibility Toolkit v2025.12
- **Standard Controls**: Migrated to new Standard UI controls introduced in the latest toolkit version
- **Component Updates**: All items now use the updated component library and patterns from FET v2025.12
- **API Compatibility**: Ensured compatibility with latest Fabric platform APIs and client libraries

### Repository Structure & Documentation
- **Repository Repositioning**: Clarified the relationship between this community repository and the official Fabric Extensibility Toolkit
- **README Restructure**: Updated documentation to clearly position this as a community tools collection rather than a replacement for the official FET
- **Support Documentation**: Updated SUPPORT.md with guidance on where to report issues (FET repo vs. this repo)
- **Sync Tooling**: Added `.github/Sync-FromFET.ps1` script to sync changes from the official FET repository while preserving custom modifications

## üîß Technical Improvements

### Fabric Extensibility Toolkit v2025.12 Integration
- **Standard Controls Migration**: Updated all UI components to use the new Standard controls from FET v2025.12
